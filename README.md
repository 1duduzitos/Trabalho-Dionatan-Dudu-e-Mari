# sistema de pedidos para lanchonete

este sistema tem como objetivo facilitar o controle de pedidos em uma lanchonete, permitindo registrar pedidos, acompanhar status e organizar entregas.

funcionalidades principais:
- cadastro de clientes
- cadastro de produtos
- criação de pedidos
- acompanhamento de status do pedido
- controle de estoque simples

metodologia utilizada:
- scrum (organização em sprints)
- kanban (trello)
- versionamento com git e github
